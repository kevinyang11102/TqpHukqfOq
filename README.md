# 酒店管理系统 java986

## 项目概述

酒店管理系统是一款基于B/S架构和MVC设计模式的前后端分离系统，采用Java和Vue技术栈开发，具备用户友好、功能全面的特点。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- 技术栈：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue
